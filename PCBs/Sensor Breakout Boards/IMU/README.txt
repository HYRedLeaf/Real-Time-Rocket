Breakout Board for the InvenSense IMU (ICM-20649).
Plugged into the I2C 1/2 Sensor Comm Port
Designed in EAGLE by Clayton Ramsey

Pin Description:
1. V (3v nominal)
2. Unused
3. SDA - input/output. I2C serial data.
4. FSYNC - input. 
5. SCL - input. I2C clock signal.
6. INT1 - output. Interrupt 1.
7. Unused
8. INT2 - output. Interrupt 2.
9. NCS - input. Chip select (SPI mode)
10. SDO - input. Serial Data Output (SPI mode) or I2C slave address.
11. GND
12. Unused
13. Unused
14. Unused
15. Unused
16. Unused
