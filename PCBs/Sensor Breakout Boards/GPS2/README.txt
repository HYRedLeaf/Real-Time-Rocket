Breakout Board for the Antenova GPS (M20050-1).
Plugged into the UART 1/2 Sensor Comm Port
Designed in EAGLE by Clayton Ramsey


Pin Description:
1. V (3v nominal)
2. Unused
3. Tx - output. Transmitter signal for UART.
4. Unused
5. Rx - input. Receiver signal for UART.
6. HW_R. System reset pin, active low
7. Unused
8. TM. Time markout, 1 P/s 
9. Unused
10. FIX - Indicates when a GPS fix is obtained
11. GND
12. HW_S. Used to enable standby mode.
13. Unused
14. Unused
15. Unused
16. Unused
